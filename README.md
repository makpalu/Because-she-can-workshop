# ✨ Because She Can – Quotes Gallery

A beginner-friendly open source project to celebrate the voices and wisdom of women in tech, leadership, and beyond — one quote at a time.  
Participants contribute by adding one inspirational quote to a shared gallery.

![Screenshot of Quotes Gallery](/screenshot.png)

---

## 🌟 What is this project?

This is a collaborative learning project for beginners to practice using Git and GitHub by:

- Forking a repository
- Making small changes
- Submitting a Pull Request (PR)

---

## 🧠 How to Contribute

Follow the steps below to add your own quote to the gallery:

### 1. **Fork the Repository**

Click the `Fork` button on the top right of this page to create your copy.

### 2. **Clone Your Fork**

Copy the link from your fork and run this in your terminal:

```bash
git clone https://github.com/YOUR-USERNAME/Because-she-can-workshop.git
cd Because-she-can-workshop
```
